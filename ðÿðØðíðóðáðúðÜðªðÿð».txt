ШАГ 1. Скачайте архив

ШАГ 2. Зайдите в панель Reg.ru
------------------------------
1. Откройте браузер на планшете
2. Зайдите в панель управления Reg.ru (туда, где вы покупали хостинг)
3. Найдите раздел «Файловый менеджер» или «Менеджер файлов"
4. Откройте папку public_html (или www, или сайт)

ШАГ 3. Загрузите файлы
----------------------
1. Удалите ВСЁ, что там есть (старые файлы WordPress)
2. Нажмите «Загрузить файлы» (или перетащите)
3. Загрузите все 6 файлов из архива
4. Готово! Сайт работает по вашему домену

Чтобы поменять текст или телефон:
- Напишите мне, что поменять
- Я пришлю обновлённый файл
- Вы загрузите его поверх старого

================================================================================
ВАРИАНТ Б: WordPress + Gutenberg (сложнее, но возможно)
================================================================================

Если всё-таки хотите WordPress:

ШАГ 1. Вставьте CSS в WordPress
--------------------------------
1. Зайдите в wp-admin (ваш-сайт.ru/wp-admin)
2. Левое меню → «Внешний вид» → «Настроить»
3. Найдите «Дополнительные CSS» (внизу списка слева)
4. Откроется поле для CSS-кода
5. Скопируйте ВЕСЬ текст из файла premium_style.css
6. Вставьте в это поле
7. Нажмите «Опубликовать» (вверху)

ШАГ 2. Создайте страницу в Gutenberg
------------------------------------
1. Левое меню → «Страницы» → «Добавить новую"
2. Название: Главная
3. Начинайте добавлять блоки («+»):

Для Hero (первый экран):
- Блок «Обложка» → фон градиент (цвета: #0f1117 → #1a1f2e)
- Внутри: Заголовок H1 + Текст + Кнопки

Для секций:
- Блок «Группа» → фон белый или #f8f6f3
- Внутри: Заголовок H2 + Текст + Колонки

Для карточек:
- Блок «Колонки» → 3 колонки
- В каждую: Заголовок H3 + Текст

Для CTA:
- Блок «Группа» → фон #c75b39
- Внутри: Заголовок H2 + Кнопки

ШАГ 3. Добавьте CSS-классы к блокам
------------------------------------
Это нужно, чтобы блоки выглядели премиально.

1. Тапните на блок (например, на «Группу")
2. Справа появится панель настроек
3. Найдите «Дополнительно» → «Дополнительный CSS класс"
4. Впишите класс:
   - Для Hero-обложки: hero-premium
   - Для секции: section-premium
   - Для карточки: feature-card-premium
   - Для CTA: cta-section-premium

НО: на планшете эта панель может быть неудобной.
